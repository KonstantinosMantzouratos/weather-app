Λίγα λόγια για το project

Ονομάζομαι Κωνσταντίνος Μαντζουράτος και αυτή είναι η πρώτη μου web εφαρμογή! Αυτή η σελίδα δημιουργήθηκε για να δείχνει ζωντανά την πρόγνωση του καιρού για τις επόμενες 3 ημέρες στην Αθήνα και τη Θεσσαλονίκη, αντλώντας δεδομένα σε πραγματικό χρόνο. Χρησιμοποιεί το Open-Meteo API για να παρέχει ακριβείς και ενημερωμένες πληροφορίες σχετικά με τη θερμοκρασία και τις καιρικές συνθήκες.
